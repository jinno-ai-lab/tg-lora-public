## 主要実装項目

### メイン学習ループ 🟡

**信頼性**: 🟡 *test_tg_lora_workflow.py・README Quick Start の拡張*

`test_tg_lora_workflow.py` の `test_multi_cycle_convergence` を本番対応に拡張:

1. HF モデル + PEFT LoRA 初期化
2. DataLoader（HF datasets）統合
3. サイクルループ:
   - `controller.propose()` → Proposal 取得
   - K ステップの勾配計算（pilot phase）
   - DeltaTracker + Velocity 更新
   - RollbackManager で保存
   - LayerSampler + Extrapolator で外挿
   - Eval → accept/rollback
   - CycleState 更新
   - TrajectoryAnalyzer で early stop 判定
   - `adapt_to_convergence` / `adapt_to_acceleration` 適用
4. 終了時サマリー出力

**実装ファイル**: `scripts/train_tg_lora.py`

### CLI インターフェース 🔵

**信頼性**: 🔵 *argparse 標準パターン*

```bash
python scripts/train_tg_lora.py --config configs/train.yaml
```

argparse で `--config`（必須）、`--resume`（チェックポイントパス）、`--output-dir`（上書き）を受け付ける。

**実装ファイル**: `scripts/train_tg_lora.py`

### Eval コールバック 🟡

**信頼性**: 🟡 *設計文書の eval フローに基づく*

pilot 損失と外挿後損失の計測。Hugging Face モデルでの forward pass による損失計算。設定可能な eval 頻度。

**実装ファイル**: `scripts/train_tg_lora.py`
