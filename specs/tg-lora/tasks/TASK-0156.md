## 背景（前 iter までの verdict の残る外部妥当性 gap）

前 iter まで（`49d36d8` → `32f128d` → `e8abf47`）で homogeneous leg の real 9B §4 A/B verdict は **non-thin SURPASSES（4 seeds/arm）+ direction-isolated（P0 confound closure）** に確定した。しかし deposit の provenance を監査すると、**学習 regime が memorization** だった: candidate arm の `last_train_loss ≈ 5e-4`、control arm は `0.0`（8 train example を 20 step で cycle → LoRA が train CE を完全に暗記）。このとき held-out valid_loss（~1.626）は**凍結 base model が支配**し、overfit adapter がわずかに perturb した値に過ぎない。すなわち headline の SURPASSES は**暗記した model 上で測定**されており、「実 SFT（generalization）でも output-first freeze が優れるか」という §4 の本来の問いへの**外部妥当性が未検証**だった。これは invented ではなく、deposit の `last_train_loss` が文字通り ~0 であること直接的に示される。

**本 iter の問題意識（feedback の原則を抽出した SINGLE highest-leverage 実証）**: AI-Hub feedback は「9B verdict を出せ」を繰り返すが、verdict は既に 3 iter にわたり取得済（feedback は stale）・REAL-Gemma/score_delta/funnel thread は本 mirror で phantom（grep 再確認・`[[ai-hub-feedback-infra-vs-this-repo]]`）。feedback の**原則**（peripheral polish 禁止・real product behavior・計測器の trustworthiness）だけを抽出し、verdict 計測器の**残る薄さ**（= memorization regime という外部妥当性 gap）を閉じる。完全 §4 verdict は private `src.data` + heterogeneous×generalize leg（>12GB）で物理 block されたままなので、**GPU が unblocked な今取り得る最大の実証**は「verdict が memorization artifact でないことの証明」= generalization regime での verdict 再取得である。

## 実装

### 1. `final_ce_train_loss` 診断 field（`scripts/run_freeze_validloss_ci_9b.py` `arm_valid_loss_9b`）

各 arm の訓練終了後、train set 上の**平均 full cross-entropy** を `eval_loss_9b(model, train_batches)` で計測し provenance に記録。これは memorization-vs-generalization の**機械可読な regime 診断**である。従来の `last_train_loss` は最終 optimizer step の loss = 凍結後は境界 activation-matching **local loss**（構造上 ≈0・arm 間で比較不能）であり、reader が「~0 = 暗記」と誤読する余地があった。`final_ce_train_loss` は train example 上の正味 CE で、≈0 なら暗記・valid と同程度なら汎化を意味する。additive（`result_to_json` は provenance を verbatim pass-through）なので既存 deposit/replay test は非破壊。

### 2. generalization-regime run（`make freeze-validloss-ci-9b-generalization`）

同一 A/B（candidate vs surrogate vs direction-control）を **generalization regime**（train_examples=48, total_steps=96 = 2 epoch・各 example を ~2x・train CE は 0 に崩壊しない）で再実行し、新規 deposit `tests/fixtures/freeze_validloss_ci_9b_generalization.json` へ（既存 2 deposit は上書きしない）。3 seeds/arm × 3 arm = 9 arm（non-thin）。研究問い: SURPASSES は memorization regime を脱いでも**生き残るか**、それとも memorization artifact か。

### 3. 受け入れ test（`tests/test_run_freeze_validloss_ci_9b.py`・+10・mutation 証明）

- `TestResultToJson::test_final_ce_train_loss_passes_through_provenance`（+1）— field が additive に deposit へ surface することを pin。
- `TestGeneralizationRegimeDeposit`（+9）— regime 診断（`final_ce_train_loss > 0.5`・train CE ≈ valid CE）/ `final_ce` vs `last_train_loss` の >1000x disambiguation / 両 verdict の replay faithfulness / monotone ranking の regime-robustness / effect-size 縮小（cross-deposit）/ direction isolation の再現 / non-thin。

## 計測結果（controlled A/B・12GB RTX 3060・`Qwen/Qwen3.5-9B` 4-bit nf4・suffix-only `last_25_percent`）

### generalization regime（本 iter・48 train / 32 valid / 96 step = 2 epoch）

| arm | order | frozen layers | valid_loss mean | **final_ce_train_loss** | train-valid gap |
|---|---|---|---|---|---|
| **candidate** | output-first | {29,30,31} | **1.5152** | **1.5071** | 0.008 |
| surrogate | random | {25,28,31}/{26,28,29}/{24,27,28} | 1.5270 | 1.5392 | 0.012 |
| control | input-first | {24,25,26} | 1.5375 | 1.5591 | 0.022 |

- **surrogate A/B**（candidate output-contig vs random）: **SURPASSES**・point **0.0118**・CI[95%] **[0.0098, 0.0137]**・non-thin（3/arm）・significant + material。
- **direction A/B**（candidate output-contig vs control input-contig・contiguity 固定）: **SURPASSES**・point **0.0223**・CI[95%] **[0.0218, 0.0227]**・non-thin。
- **regime 診断**: 全 arm の `final_ce_train_loss` は ~1.5（**memorization regime の ~5e-4/0.0 と対照的**）。train CE ≈ valid CE（gap ≤0.022）= **真の汎化**（memorization では gap ~1.6）。`last_train_loss` は ~1.7e-4（local loss）→ `final_ce` との比 **8775x** = 両者は別物の証。

### regime-robustness verdict（cross-deposit）

| regime | candidate_mean | surrogate_mean | point_improvement | verdict |
|---|---|---|---|---|
| memorization（`freeze_validloss_ci_9b_surrogate.json`）| 1.6255 | 1.7698 | 0.1443 | SURPASSES |
| **generalization（本 iter）** | **1.5152** | **1.5270** | **0.0118** | **SURPASSES** |

- **方向は regime を通じて robust**: 両 regime で candidate < surrogate < control の単調順位が保持（SURPASSES → SURPASSES）。
- **大きさは regime-dependent**: effect size は **12.2x 縮小**（0.144 → 0.012）。verdict の*方向*は頑健、*大きさ*は regime 依存、という**正直な計測**（pre-decided でない）。
