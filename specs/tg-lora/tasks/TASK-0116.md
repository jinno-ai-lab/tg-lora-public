## 主要実装項目

### 1. 感度分析スクリプト (scripts/analyze_sensitivity.py) 🟡

**信頼性**: 🟡 *summarize_sweep.pyの分析拡張として推測*

- `load_sweep_results(run_dir: str) -> list[dict]`: スイープ結果読込
- `compute_correlation_matrix(results: list[dict], params: list[str], metrics: list[str]) -> dict`: 相関行列計算
- `rank_sensitivity(correlations: dict) -> list[tuple[str, float]]`: 感度ランキング
- `generate_sensitivity_report(results: dict, output_path: str) -> None`: JSON レポート出力
- CLI: `python scripts/analyze_sensitivity.py --run-dir runs/sweep --output sensitivity_report.json`
