## 主要実装項目

### 1. 設定スキーマ定義 🔵

**信頼性**: 🔵 *既存YAML構造の完全読み込みより*

```python
# src/training/config_schema.py
from pydantic import BaseModel, Field, field_validator

class ExperimentConfig(BaseModel):
    name: str
    seed: int = Field(ge=0)

class ModelConfig(BaseModel):
    name_or_path: str
    dtype: str = "bfloat16"
    load_in_4bit: bool = True
    bnb_4bit_quant_type: str = "nf4"
    bnb_4bit_compute_dtype: str = "bfloat16"
    device_map: str = "auto"

class LoRAConfig(BaseModel):
    r: int = Field(ge=1)
    alpha: int = Field(ge=1)
    dropout: float = Field(ge=0.0, lt=1.0)
    target_modules: str = "all-linear"

class DataConfig(BaseModel):
    train_path: str
    valid_quick_path: str
    valid_full_path: str
    gold_test_path: str = ""
    max_seq_len: int = Field(ge=1, default=2048)

class TrainingConfig(BaseModel):
    batch_size: int = Field(ge=1)
    grad_accumulation: int = Field(ge=1)
    learning_rate: float = Field(gt=0.0)
    weight_decay: float = Field(ge=0.0, default=0.0)
    max_steps: int = Field(ge=1, default=1500)
    max_cycles: int = Field(ge=1, default=500)
    gradient_checkpointing: bool = True
    max_grad_norm: float = Field(gt=0.0, default=1.0)
    warmup_steps: int = Field(ge=0, default=50)
    save_every_steps: int = Field(ge=1, default=250)

class EvalConfig(BaseModel):
    quick_eval_examples: int = Field(ge=1, default=64)
    full_eval_every_steps: int = Field(ge=1, default=250)
    full_eval_every_cycles: int = Field(ge=1, default=10)
    rollback_tolerance: float = Field(ge=0.0, default=0.005)
    save_predictions: bool = True

class LoggingConfig(BaseModel):
    backend: str = "mlflow"
    log_every_steps: int = Field(ge=1, default=10)
    log_every_cycles: int = Field(ge=1, default=1)
    save_every_cycles: int = Field(ge=1, default=25)
    run_dir: str

class TGLoRAParams(BaseModel):
    K_initial: int = Field(ge=1)
    K_candidates: list[int]
    N_initial: int = Field(ge=1)
    N_candidates: list[int]
    alpha_initial: float = Field(gt=0.0)
    alpha_min: float = Field(gt=0.0)
    alpha_max: float = Field(gt=0.0)
    alpha_log_sigma: float = Field(gt=0.0, default=0.15)
    beta_initial: float = Field(gt=0.0, lt=1.0)
    beta_candidates: list[float]
    relative_update_cap: float = Field(gt=0.0, lt=1.0)
    active_layer_strategy: str
    random_middle_layers: int = Field(ge=0, default=2)

    @field_validator("alpha_min", "alpha_max")
    @classmethod
    def alpha_range(cls, v, info):
        # validated in model_validator
        return v

class BaselineConfig(BaseModel):
    experiment: ExperimentConfig
    model: ModelConfig
    lora: LoRAConfig
    data: DataConfig
    training: TrainingConfig
    eval: EvalConfig
    logging: LoggingConfig

class TGLoRAConfig(BaselineConfig):
    tg_lora: TGLoRAParams
```

**実装ファイル**: `src/training/config_schema.py`（新規）

### 2. load_and_validate_config関数 🔵

**信頼性**: 🔵 *既存main()関数のOmegaConf.loadパターンより*

```python
def load_and_validate_config(config_path: str | Path) -> BaselineConfig | TGLoRAConfig:
    raw = OmegaConf.load(config_path)
    data = OmegaConf.to_container(raw, resolve=True)
    if "tg_lora" in data:
        return TGLoRAConfig(**data)
    return BaselineConfig(**data)
```

**実装ファイル**: `src/training/config_schema.py`

### 3. テスト 🔵
