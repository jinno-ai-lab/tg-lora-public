## 主要実装項目

### 1. temperature パラメータフロー統合テスト 🔵

**信頼性**: 🔵 *既存layer_sampler.pyテストとconfig_schemaテストのパターンより*

mock-based で config→TGLoRAParams→select_active_layers の温度パラメータ伝播を検証。3つの温度値でレイヤー選択分布の変化を確認。

**実装ファイル**: `tests/test_layer_sampler.py`

### 2. 境界値テスト 🟡

**信頼性**: 🟡 *temperature=0や極端に大きな値の挙動は実装依存のため推測*

temperature が 0 に近い場合（集中選択）、非常に大きい場合（均一選択）のエッジケーステスト。NaN/Inf 入力の防御確認。

**実装ファイル**: `tests/test_layer_sampler.py`

## 実装手順

1. TGLoRAParams の temperature フィールド定義を確認
2. train_tg_lora.py から select_active_layers への temperature 渡し箇所を確認
3. mock を用いて config→TGLoRAParams→select_active_layers のパラメータ伝播テストを実装
4. temperature=0.1, 1.0, 5.0 の3パターンでレイヤー選択分布を比較するテストを追加
5. temperature=1e-6（0に近い）、temperature=1e6（非常に大きい）の境界値テストを追加
6. NaN/Inf 入力時にエラーまたは適切なフォールバックが行われることを確認
7. pytest で全テストパス確認
