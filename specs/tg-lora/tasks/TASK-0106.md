## タスク概要

Stage 2 multi-seed replicationを実行し、Gate G1のpass/failを確定する。1回限りのsmokeではなく、seeds 42/43/44 でTG vs BLの再現性を検証する。

## 依存タスク

- **前提タスク**: [TASK-0105: Paper Gate評価自動化](TASK-0105.md)
- **後続タスク**: Stage 3 (G2 frontier sweep), Stage 4 (G3 external quality)

## 完了条件

- [ ] canonical seed-42 runがGate G0 (Hygiene) をpassする
- [ ] multi-seed (42/43/44) runが完了しaggregate_summary.jsonが生成される
- [ ] `make paper-memory-evaluate-gates` でGate G0/G1のpass/failが判定される
- [ ] G1 pass時: Stage 3 (frontier sweep) へ進む判断が記録される
- [ ] G1 fail時: C0への格下げ判断と理由がpaper_experiment_plan.mdに記録される
- [ ] paper_results_snapshot.mdにStage 2結果が追加される（write-once per round）

---

## 実行手順

### Step 1: Canonical seed-42 run

```bash
make paper-memory SEEDS='42' TARGET_BP=240 MAX_SEQ_LEN=1024 \
    OUTPUT_BASE=runs/paper_memory_suite_seed42_canonical
```

G0検証:
```bash
make paper-memory-evaluate-gates \
    GATE_SUMMARY=runs/paper_memory_suite_seed42_canonical/aggregate_summary.json \
    GATE_SKIP=G2 G3 G4
```

### Step 2: Multi-seed replication

```bash
make paper-memory SEEDS='42 43 44' TARGET_BP=240 MAX_SEQ_LEN=1024 \
    OUTPUT_BASE=runs/paper_memory_suite_ms3_s1024
```

### Step 3: Gate G1 evaluation

```bash
make paper-memory-evaluate-gates \
    GATE_SUMMARY=runs/paper_memory_suite_ms3_s1024/aggregate_summary.json \
    GATE_OUTPUT=runs/paper_memory_suite_ms3_s1024/gate_report.json \
    GATE_SKIP=G2 G3 G4
```

### Step 4: Decision recording

- G1 pass → Stage 3 (MAX_SEQ_LEN=1536,2048,3072 frontier sweep) へ
- G1 fail → C0格下げ、paper_experiment_plan.mdに記録
