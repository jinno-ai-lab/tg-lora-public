## 主要実装項目

### 1. 統計分析モジュール (src/analysis/stats.py) 🟡

**信頼性**: 🟡 *paper_experiment_plan.md の統計要件から推測*

- `confidence_interval(data: list[float], confidence: float = 0.95) -> tuple[float, float, float]`: 平均・下限・上限を返却
- `paired_t_test(baseline: list[float], treatment: list[float]) -> tuple[float, float]`: t統計量・p値を返却

- `analyze_multi_seed(aggregate_summary: dict) -> dict`: 全メトリクスの統計サマリを生成

**実装ファイル**: `src/analysis/stats.py`, `src/analysis/__init__.py`
