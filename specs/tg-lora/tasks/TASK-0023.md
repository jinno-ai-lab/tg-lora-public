## 主要実装項目

### 1. CI ワークフロー作成 🟡

**信頼性**: 🟡 *標準的な GitHub Actions パターンより推測*

.github/workflows/test.yml を作成:
- トリガー: push (main, feature ブランチ), pull_request
- ジョブ: テスト実行 (pytest)
- ステップ: checkout → Python セットアップ → pip install → pytest --cov
- カバレッジ失敗閾値: 95% (現在99%)

**実装ファイル**: `.github/workflows/test.yml`

### 2. CI 用依存関係整理 🔵

**信頼性**: 🔵 *pyproject.toml 既存構成より*

CI 環境でのインストールを効率化するため、dev 依存関係に必要なパッケージが全て含まれていることを確認する。

**実装ファイル**: `pyproject.toml`

### 3. CI での GPU 依存テストのスキップ設定 🟡

**信頼性**: 🟡 *test_smoke.py 等 GPU 依存テストのマーカーから推測*

CI (CPU only) で実行できないテストをマーキングし、スキップ設定を追加する。@pytest.mark.gpu デコレータ等で制御。

**実装ファイル**: `tests/conftest.py`
