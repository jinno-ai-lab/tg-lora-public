## 主要実装項目

### 1. 論文結果エクスポート (scripts/export_paper_results.py) 🟡

**信頼性**: 🟡 *paper_experiment_plan.md の論文出力要件から推測*

- `load_aggregate(path: str) -> dict`: aggregate_summary.json読込
- `generate_latex_table(data: dict, stats: dict | None = None) -> str`: LaTeX形式テーブル生成
- `generate_markdown_table(data: dict, stats: dict | None = None) -> str`: Markdown形式テーブル生成
- `export_csv(data: dict, output_path: str) -> None`: CSV形式エクスポート
- CLI: `python scripts/export_paper_results.py --input aggregate_summary.json --format latex|markdown|csv [--output FILE]`

**実装ファイル**: `scripts/export_paper_results.py`
