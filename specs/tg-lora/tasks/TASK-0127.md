## 主要実装項目

### trap handler 追加 🔵

**信頼性**: 🔵 *requirements.md EDGE-003要件より*

スクリプト冒頭（MERGED_DIR変数定義後）にtrap handlerを追加:

```bash
cleanup() {
    if [ -n "${MERGED_DIR:-}" ] && [ -d "$MERGED_DIR" ]; then
        rm -rf "$MERGED_DIR"
    fi
}
trap cleanup EXIT INT TERM
```

**実装ファイル**: `scripts/run_eval_lora.sh`
