## 主要実装項目

### Frontier Sweepスクリプト 🔵

**信頼性**: 🔵 *paper_experiment_plan.md Stage 3定義より*

`scripts/run_frontier_sweep.sh`:
- SEQS引数（デフォルト: "1536 2048 3072"）を受け取り、各MAX_SEQ_LENで`make paper-memory`を実行
- 各runのaggregate_summary.jsonを収集
- baseline OOM/CUDA errorの検出（exit code またはエラーログパターン）
- TG completion確認
- frontier separation結果をfrontier_report.jsonに集約

**実装ファイル**: `scripts/run_frontier_sweep.sh`, `Makefile`

### Frontier分離検出 🔵

**信頼性**: 🔵 *paper_experiment_plan.md G2 pass条件より*

- baseline失敗（OOM/CUDA/non-completion）とTG成功のペアを検出
- 各MAX_SEQ_LENでのメモリ使用量推移を記録
- G2.3 frontier separation判定を自動化
