## タスク概要

TG-LoRA の実際の削減効果（reduction_rate）と速度向上を測定するベンチマークフレームワークを構築する。標準 LoRA ファインチューニングとの比較により、TG-LoRA の有効性を定量的に示す。

## 依存タスク

- **前提タスク**: [TASK-0004](TASK-0004.md), [TASK-0006](TASK-0006.md)
- **後続タスク**: なし（Phase 3 最終タスク）

## 完了条件

- [ ] `scripts/benchmark.py` が標準 LoRA と TG-LoRA の比較を実行
- [ ] 測定項目: wall-clock time, reduction_rate, final_loss, acceptance_rate
- [ ] 結果を Markdown テーブル形式でレポート出力
- [ ] 複数モデルサイズ（0.5B, 1.5B など）で比較可能
- [ ] ベンチマーク結果が `reports/benchmark/` に保存される

## 主要実装項目

### ベンチマークスクリプト 🟡

**信頼性**: 🟡 *ベンチマーク設計に推測を含む*

比較実験の設計:

1. **標準 LoRA**: 固定 K で全ステップ勾配計算（baseline）
2. **TG-LoRA**: 同等の等価ステップ数でサイクル実行
3. 同一データ・同一シード・同一初期重みで公平に比較
4. 複数設定（K/N 組み合わせ）でスイープ

**実装ファイル**: `scripts/benchmark.py`

### レポート生成 🔵

**信頼性**: 🔵 *MetricsRecorder（TASK-0006）の出力に基づく*

Markdown テーブルで以下を比較表示:

| 設定 | Wall-clock (s) | 等価ステップ | reduction_rate | 最終損失 | acceptance_rate |
|------|---------------|------------|---------------|---------|----------------|
| 標準 LoRA | ... | ... | 0% | ... | - |
| TG-LoRA (K=3,N=5) | ... | ... | ...% | ... | ...% |

**実装ファイル**: `scripts/benchmark.py`

### モックベンチマーク 🔵

**信頼性**: 🔵 *SimpleLoRAModel での既存テストパターン*

CI で実行可能な小規模モデルベンチマーク。CUDA なしでも動作する軽量版。

**実装ファイル**: `tests/test_benchmark.py`
